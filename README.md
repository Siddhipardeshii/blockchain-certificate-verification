# Blockchain Certificate Verification System

## What it does
Prevents certificate fraud by storing SHA-256 hash of 
certificates on Ethereum blockchain using Ganache locally.

## Tools Used
- Ganache — local Ethereum blockchain
- Remix IDE — smart contract development
- Solidity — smart contract language
- Web3.js — blockchain interaction
- HTML/CSS/JS — frontend

## Contract Address (Local Ganache)
0xf9c5d37729f4431C4370d95e323968FF67C70711

## How to Run
1. Start Ganache on http://127.0.0.1:7545
2. Open index.html with Live Server in VS Code
3. Paste contract address and click Connect
4. Upload certificate to issue or verify

## How it Works
Certificate → SHA256 Hash → Store on Blockchain → Verify anytime
